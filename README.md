# Dataset
https://challenge.kitware.com/#challenge/n/ISIC_2017%3A_Skin_Lesion_Analysis_Towards_Melanoma_Detection

To create dataset :
```
python main.py create
```

To train the classifier
```
python main.py train
```

To test the classifier
```
python main.py test
```
